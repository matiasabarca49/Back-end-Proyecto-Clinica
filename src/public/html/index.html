Endpoints Destacados

  • GET /users - Obtener todos los usuarios registrados.
    Requiere autenticación con token JWT.
  • POST /login - Iniciar sesión y obtener un token JWT.
    Envia el correo y contraseña del usuario.
  • POST /logout - Cerrar sesión y eliminar el token JWT.
    No requiere autenticación previa.

Tecnologías Utilizadas

Este backend está desarrollado con un conjunto de tecnologías modernas que garantizan seguridad, escalabilidad y rendimiento.

Node.js

Entorno de ejecución para JavaScript en el servidor.

Express.js

Framework minimalista para construir aplicaciones web y APIs.

MongoDB

Base de datos NoSQL para almacenar datos de forma flexible y escalable.

Mongoose

ODM para trabajar con MongoDB de manera más eficiente.

Bcrypt

Librería para encriptar contraseñas y aumentar la seguridad de los datos.

JWT

Sistema de autenticación basado en tokens para sesiones seguras.

Autentícate para acceder y probar los endpoints protegidos.
